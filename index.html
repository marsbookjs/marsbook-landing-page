How it works

Code, run, iterate.
All in one place.

1

Write TypeScript or JavaScript

Monaco editor with full IntelliSense. Prettier formats on save. Switch between TS and JS per cell.

2

Run and see output inline

Hit Shift+Enter. Console output, tables, markdown, images and HTML all render directly below the cell.

3

Share state across cells

Variables defined in any cell are available everywhere below it — no imports, no re-runs needed.

PM
How do I read a JSON file in my current cell and filter users by score above 80?
✦
Sure! Since you're in a Node.js notebook, use fs/promises with a top-level await:

import { readFile } from "node:fs/promises";
const data = JSON.parse(await readFile("./users.json", "utf8"));
display.table(data.filter(u => u.score > 80));

The result table will render inline below the cell. Your data variable will also be available in all cells below.
PM
What about TypeScript types for that?
✦
Add an interface above the import — MarsBook transpiles TS on the fly:

interface User { name: string; score: number; }
const users: User[] = JSON.parse(...);

AI Features

AI that knows your notebook.

The AI assistant is context-aware — it sees your current cell and can help you write, debug, and understand code without switching tabs.

Context-aware chat powered by Groq. Switch between LLaMA and OpenAI-compatible models live.

A dedicated cell type that sends a prompt to any LLM and streams the response inline, with per-cell model, system prompt, and temperature.

◈

Inline Completions

Get AI code suggestions as you type in any cell. Powered by the same Groq key — no extra setup.

Display API

Rich output, one line.

The display API renders structured content directly inside notebook cells — no HTML needed.

display.text(value)
Renders plain text output in the cell result area.
display.text("Hello world");
display.markdown(value)
Fully rendered Markdown — headings, bold, tables, code blocks, and links.
display.markdown("## Results\n- ...");
display.html(value)
Raw HTML rendered in a sandboxed iframe. Full styling support.
display.html("<b>Bold</b>");
display.image(src, alt?)
Renders an image by URL or base64 data URI directly below the cell.
display.image("https://...");
display.table(rows)
Renders an array of objects as a formatted, scrollable table with column headers.
display.table([{name:"A",val:1}]);
display({ type, … })
Object-style call — pass any type as a property for a more expressive style.
display({ type:"html", html:"…" });

Get Started

Up and running in 30 seconds.

No Python. No config. Just Node.js 20+ and one command.

🌐 Global Install Recommended
Install once, run from anywhere. The marsbook command becomes available globally on your system.
$npm install -g marsbook
$marsbook  # opens http://127.0.0.1:3113
$marsbook --port 4000 --workspace ~/notebooks
📦 Local / npx
No global install needed. Run it on-demand with npx, or add it as a local dev dependency in your project.
$npx marsbook  # no install needed
$npm install marsbook
$npx marsbook
🤖
Enable AI features
Set your Groq API key before starting to unlock the AI assistant, AI prompt cells, and inline code completions. Get a free key at console.groq.com.
$ export GROQ_API_KEY=your_key_here
$ marsbook
